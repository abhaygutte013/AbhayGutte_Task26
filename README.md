# Hello World React App

## Objective

This is a simple React application created as a practice task. It displays the text "Hello, World!" on the webpage.

## Technologies Used

- React
- Vite
- JavaScript

## How to Run

1. Open the project folder.
2. Install dependencies (if not already installed).

npm install

3. Start the development server.

npm run dev

4. Open the local URL shown in the terminal (usually http://localhost:5173).

## Output

The webpage displays:

Hello, World!

- Understood the basic folder structure.
